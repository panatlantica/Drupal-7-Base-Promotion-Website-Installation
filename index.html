Drupal-7-Base-Promotion-Website-Installation by panatlantica

A bundle of Drupal core version 7, library folder items, base themes and selected contributed modules typical for a promotion web site installation

We regularly install Drupal 7 for promotion web sites. This collection of Drupal items reflects our selection of Drupal modules, library folder items and our favorite base themes to quickly set up a site without searching for the required elements every time anew on Drupal.org
